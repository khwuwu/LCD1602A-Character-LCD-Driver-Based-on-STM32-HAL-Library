# LCD1602A-Character-LCD-Driver-Based-on-STM32-HAL-Library

This is an LCD1602A character display driver program written using the STM32 microcontroller along with the HAL function library provided by ST.

In the folder 'Driver_Function', it contains the driver for the LCD1602A display and the main.c file used with it.
